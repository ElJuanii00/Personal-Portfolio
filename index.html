Frontend Developer

Mi trabajo se enfoca en el lado del cliente, es decir todo lo que el usuario disfruta en el navegador. Para eso uso mis herramientas favoritas que uso son HTML, CSS y JavaScript donde me enfoco en diseñar y estructurar la web donde lo pueda adaptar a cualquier dispositivo o plataforma.

Git y Github

Me gusta tener mi código actualizado y que pueda gestionar las versiones del proyecto que bifurcan en el tiempo, para esto me gusta usar Git y GitHub. Git es un sistema control de versiones en el cual me facilita mover y gestionar mis archivos en cualquier momento y poder compartir mi código a la comunidad en GitHub.

PostCSS

Es herramientas que uso para optimizar mi tiempo escribiendo código en CSS es usar el poder que tiene PostCSS, es decir me ayuda a transformar mi código CSS gracias a los diversos plugins de JavaScript que me ayudan a completar mi trabajo.

WPO

Implemento Critial CSS a tu página web para optimizar el proceso de carga del navegador llegando a un resultado de valoración superior de 90/100 en PageSpeed. ¿Y esto para qué? Esto sirve para que Google posicione tu página web en los resultados de búsqueda superiores.

Sistema de Diseño

Cada proyecto que realizo un sistema de diseño documentando cada detalle de lo que se va a crear adaptándome mejor a las necesidades del cliente. Para esto uso Notion como una herramienta de trabajo que me ayuda a diseñar un sistema de diseño del proyecto para ahorrar tiempo y ser más consistente en el objetivo principal.

Prototipos

Una manera de generar mis ideas de cómo va a aparentar la estructura y estilos que tendrá finalmente el sitio web es crear un prototipo. A lápiz y papel creo los wireframe como la estructura del contenido y con la ayuda de Adobe XD le doy vida a las ideas estilizando el contenido del proyecto.
